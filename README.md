此款卡刀宏具备的功能：

1、支持鼠标键盘上的绝大多数键位

2、支持更多技能键位设置

3、支持组合键，如魔兽世界等网游中的CTRL+1或SHIFT+1或ALT+1等需要组合键位释放技能

4、支持取色，模拟鼠标移动点击等操作

5、卡刀数据云存储，重要的卡刀数据不丢失

6、安全无毒，无后门，已经过微软软件绿色认证

7、免费！免费！免费！

技术栈：
本宏软件技术栈采用的是C#原生开发。没有基于按键精灵、AHK等第三方软件平台或工具进行开发。
